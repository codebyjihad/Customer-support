# Customer Support App

Customer Support App is a modern web application to manage customer support tickets. Tickets can be marked as **In Progress** or **Resolved**, and the app dynamically displays the counts for each status. Users can update ticket status with a single click.

---

## Features

- Track tickets as **In Progress** or **Resolved**  
- Dynamic count display of In Progress and Resolved tickets  
- Update ticket status with a single button click  
- Responsive UI built with TailwindCSS and DaisyUI  
- Notifications using React Toastify  

---

## Tech Stack

- **Frontend:** React + Vite  
- **Styling:** TailwindCSS + DaisyUI  
- **Notifications:** React Toastify  
- **Icons:** React Icons  
- **Linting:** ESLint
